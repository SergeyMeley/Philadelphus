# Avalonia migration docs

Финальный статус: активных работ по миграционному техдолгу Avalonia в этой папке не осталось.

Канонический актуальный документ:
- `15-avalonia-tech-debt.md` — финальное состояние после закрытия разделов A, D-M и smoke QA по H.

Документы `00-*` - `14-*` являются историческими снимками анализа, roadmap и handoff-сессий. В них сохранены старые формулировки вроде "открыто", "нужно", "осталось" и чекбоксы этапов; они не являются текущим списком задач. Для актуального решения по таким пунктам смотреть `15-avalonia-tech-debt.md`.

Оставшиеся темы вынесены за рамки этой миграционной задачи:
- формулы и редактирование формульных значений: `docs/attribute-value-formula/`;
- полный async-рефакторинг Core-уведомлений;
- удаление старого WPF-проекта;
- обновление библиотек расширений;
- нагрузочные инфраструктурные проверки Kafka;
- продуктовая полировка финальных SVG-иконок.
