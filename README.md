# Проект Philadelphus / Чубушник
## Описание проекта
Приложение Чубушник разрабатывается как альтернативный вариант решения различных задач в приложениях по типу MS Excel.
Основное отличие от Excel в том, что оперировать здесь мы будем не листами и таблицами, а классами и объектами в соответствии с лучшими практиками ОО(П).
Наверное, каждый пользователь сталкивался со сложностями при добавллении в табличный редактор иерархических данных. Каждый сталкивался с непониманием, когда следует разбить данные на разные таблицы для исключения дублирования и расхождения данных, а когда объединить для упрощения и ускорения поиска без сложных формул. Сложности с преобразованием иерархических данных в плоские таблицы окупается быстротой и удобством массовой работы с большим количеством объектов, простотой и низким порогом вхождения для специалистов разных сфер деятельности. Мы попробуем сохранить все плюсы MS Excel, но привнести дополнительные бонусы и удобные фишки.

Приложение размещается в виде zip-архива, поэтому установка не требуется, достаточно просто распаковать архив в любую требуемую директорию и запустить exe-файл.
Для работы также нужно создать базу данных, на данный момент поддерживается только PostgreSQL.
## Основные возможности
- создание иерархических структур данных;
- реализация основных принципов ООП - инкапсуляции, наследования, полиморфизма, абстракции;
- добавление пользовательских атрибутов;
- автоматическая фиксация основной информации для проведения аудита (кто / когда создал, кто изменил удалил и др.);
- возможность применения различных адаптеров для хранения информации в различных хранилищах, таких как БД (например, PostgreSQL) или файловые хранилища (например, JSON-файлы);
- возможность объединения различных данных из различных хранилищ в единые рабочие пространства - репозитории;
- логирование;
- и многое другое.
